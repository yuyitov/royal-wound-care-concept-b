# Royal Wound Care — Maqueta Estática (Concepto) — VERSIÓN B

⚠️ Este es el directorio de la VERSIÓN B. No modificar el directorio de la Versión A (`royal-wound-care-concept`).
- Versión A (congelada): https://yuyitov.github.io/royal-wound-care-concept/
- Versión B (activa): https://yuyitov.github.io/royal-wound-care-concept-b/

Este proyecto es una maqueta externa segura y estática para Royal Wound Care.
No es el sitio real. No está conectado a ningún sistema de producción.

## Reglas de trabajo

- No tocar el sitio real en GoHighLevel.
- No modificar el dominio real.
- No conectar formularios reales.
- No capturar datos de pacientes.
- No incluir PHI ni información sensible.
- No agregar tracking, analytics, backend, base de datos, npm, React ni scripts externos.
- La maqueta debe mantenerse como un solo archivo estático `index.html`, con CSS interno y JavaScript mínimo.
- El proyecto se publicará en un repositorio nuevo y separado de GitHub usando GitHub Pages.

## Proceso para cambios

1. Antes de hacer cambios visuales o de contenido, revisar el HTML actual.
2. Proponer un checklist de cambios al usuario.
3. No modificar `index.html` hasta que el usuario apruebe el checklist.
